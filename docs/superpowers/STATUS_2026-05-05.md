## LME 가격 출처 이중화 (이번 세션 신규)

- **Primary**: NH선물 PDF (full schema — OHLC, 3m, bid/ask, OI, settlement, forwards, monthly_avg, inventory breakdown, SHFE)
- **Fallback**: westmetall.com (minimal — settlement.cash, settlement.3m, inventory.current)
- **Cross-validation 결과**: 6 metals × 200 dates × 3 fields = 3,600 비교 → **Δ=0.00 완전 일치**
- **Backfill**: 2008~2014 westmetall로 8년 추가, 1,925개 synthetic JSON
- **자동 fallback**: KR 공휴일 NH 미발행 시 collect.yml이 westmetall로 자동 보완 (`builder.lme_backfill --mode fallback`)

## 뉴스 파이프라인 (Phase 1b 완료)

- **Active sources**: mining.com (RSS, 36 entries/run), 철강금속신문 snmnews (RSS, 50/run), 머니투데이 (RSS, 100/run), GDELT 2.0 (API, 75/run)
- **Inactive (defer)**: nonferrous.or.kr (CI TCP timeout, "never" decision), KORES (dead site), KOMIS (JS-rendered)
- **LLM**: Gemini 2.5 Flash with `response_json_schema` 강제 (object enum). null=0 가드.
- **Failover chain**: Gemini → (groq/cerebras 미구성) → fail-soft raw 저장
- **Silent-fail guard**: scraper 0건 또는 5+ 입력 100% null 시 `exit(2)` → CI 알림
- **Schema**: `[date, source, url, url_hash, title, summary_ko, metals, sentiment, event_type, confidence, lang]`
